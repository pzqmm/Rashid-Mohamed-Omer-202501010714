# Food Delivery System - Backend Receipt Generator

## 4.1. Purpose of the application
This application is a simple backend component for a Food Delivery System. It collects customer order details (name, food item, quantity, price, and delivery preference) and generates an itemized receipt that calculates the subtotal, includes a 5% service charge, adds delivery fees if applicable, and computes the grand total.

## 4.2. Tech Stack
* Python 3

## 4.3. How to use
1. Open your terminal or command prompt.
2. Navigate to the `week_8` directory.
3. Run the application using the command: `python main.py`
4. Follow the on-screen prompts to enter the customer details and order information.
