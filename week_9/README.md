# IT Helpdesk Ticket Registration System

## 1. Purpose of the application
A simple modular Python program designed for the City University Malaysia IT Support team to register and manage technical issues reported by students.

## 2. Tech Stack
* Python 3

## 3. How to use
1. Run the main.py file in your terminal.
2. Enter the student's details and the issue description.
3. Specify the priority level (High/Medium/Low).
4. The system will automatically assign a technician and print the final ticket.
